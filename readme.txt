Easily display CodeGrape items inside WordPress widget.

== Description ==

A simple and powerful WordPress plugin with which you can display CodeGrape items as a WordPress widget. Several smart options are provided for selecting and ordering. You can select CodeGrape latest items or items from one or more specific users. Optionally, you can connect items with your affiliate links as well.

If you are a CodeGrape author this widget is perfect for you.

### Features

* Using CodeGrape official API
* Supports all CodeGrape marketplace categories
* Select latest CodeGrape items
* Select items from one or more specific users
* Order items by release date, upload date, numbers of sales, price or random
* Add your affiliate links to CodeGrape items
* Caching system integrated for better performance

== Installation ==

1. Upload `smartcodegrapewidget.zip` to plugins via WordPress admin panel or upload unzipped folder to your `wp-content/plugins/` folder
2. Activate the plugin through the `Plugins` menu in WordPress
3. Go to `Appearance -> Widgets` to add and setup the widget
